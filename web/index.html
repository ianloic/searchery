AwesomeSearch
Web search results in your Firefox 3 awesome bar

AwesomeSearch is a new Firefox extension which brings web search into Firefox 3's new Awesome Bar. As you type into your address bar Firefox will search your history for pages that match and AwesomeSearch will search the web.

So when you want to find something, just start typing. If you've been there before it will show up at the top of the list. If you haven't then you can find it in the web search results that show up in blue.

AwesomeSearch is built on top of new features only available in Firefox 3. To try it out you'll have to install a beta of Firefox 3 first. Don't worry, it's getting pretty stable.

AwesomeSearch is a Firefox extension. It depends on features that are unique to Firefox. There are no plans to port it to other web browsers. Why don't you download Firefox?
